# Ejercicio 2

Completa el script `inversorDeObjetod` para que dado un objeto el script debe invertir sus keys y valores volviendo los keys values y los values keys

Si la entrada es el objeto

```javascript
 {
  uno: 'a',
  dos: 'b',
  tres: 'c'
 }
```
La salida debe ser:

```javascript
  {
    a: 'uno',
    b: 'dos',
    c: 'tres'
  }
```
