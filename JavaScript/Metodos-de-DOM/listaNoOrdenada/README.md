# Lista no ordenada

Crea un script que construya una lista no ordenada a partir de un array.


### Array
```javascript
['Charles', 'Leo', 'Isra', 'David']    
```

### HTML esperado
```HTML
<ul>
  <li>Charles</li>
  <li>Leo</li>
  <li>Isra</li>
  <li>David</li>
</ul>
```
